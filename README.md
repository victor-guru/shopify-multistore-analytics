📌 Project Overview
This dashboard analyzes sales performance, customer behavior, and product trends using Power BI.

🎯 Objectives
- Track total revenue
- Analyze sales by product
- Identify top-performing products
- Monitor order status (Delivered, Pending, Cancelled)

## 📂 Dataset
- Source: Sample E-commerce dataset
- Includes: Orders, Revenue, Payment Methods, Customer Data

## 🛠 Tools Used
- Power BI
- Excel

 📈 Key Insights
- Highest revenue comes from tech products (Printer and Laptop).
- Only 39% of orders are successfully completed. That means:
61% of orders fail or are incomplete
- Coupon usage impacts sales positively

Executive Summary 
The dashboard shows that while revenue generation is strong, the business suffers from a low conversion rate (39%) and high cancellation and return rates. Key products like printers and laptops drive revenue but also contribute significantly to cancellations, indicating operational or fulfillment issues. Additionally, sales trends are inconsistent, suggesting a need for better demand forecasting and customer retention strategies.
